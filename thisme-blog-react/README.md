# thisme-blog-react 

#### 如何安装

``` txt
#第一步
yarn 或者 npm install
#第二步
自我整理目录结构(雾), 本项目采用TypeScript + React Hook 的方式, 并在里面存有例子,确保会用Tsx
```

#### 指令列表

| 名称             | 作用                                                        |
| ---------------- | ----------------------------------------------------------- |
| ```yarn start``` | 启动devServer,默认端口8888                                  |
| ```build:dev```  | 项目的dev模式,是以prod模式构建(入口**src/index.tsx**)       |
| ```build:prod``` | 项目的prod模式,是以prod模式构建(入口**src/prod/index.tsx**) |
