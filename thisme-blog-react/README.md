# thisme-blog-react 

> 对[\<simple-react-app\>](../simple-react-app)的扩展, 在运行前保证你有```node``` 和```yarn```, 
> 更是我要完成的小站目标,有兴趣点个star  , 由于我的```webpack```和```webpack-dev-server```
> 是全局安装的, 请自行安装

#### 指令

* ```yarn start``` 启动服务默认端口8888
* ```yarn build``` 开发者模式构建

#### 技术闸

* React Hooks
* Redux
* Router
* .....

#### 目录

* build 构建生成得文件
* config ```webpack```配置文件
* node_modules 库文件
* src 核心代码
  * component 基础组件
  * editor 编辑器组件
  * img 图片文件夹
  * menu 菜单文件
  * redux 状态组件
    * reducers  主要状态目录
  * router 路由组件
    * page 页面组件
  * style 样式文件夹
  * test 测试文件夹
  * util 工具文件夹
* static 静态文件放这里
