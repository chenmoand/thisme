### 如何下载 && 安装

```txt
#第一步 [克隆项目]
> git clone https://github.com/chenmoand/thisme.git 
#第二步 [安装前置 二选一]
> yarn deploy:yarn 或者 npm deploy:npm
#第三步 [打开IDEA]
File -> open
#第四步 [It's up to you]
```
