###  Command

* 常用指令

  ```yarn web``` 将thisme-blog-react构建并且提交githubPage

  ```yarn ci``` 一个强大的自动部署指令, 将thisme-blog-react构建并复制到thisboot上

* 一次性指令(2选一)

  ```yarn deploy:yarn```  用yarn的方式下载项目所需要的依赖
  
  ```npm deploy:npm``` 用npm的方式下载项目所需要的依赖
